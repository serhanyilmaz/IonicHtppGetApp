1.)LoginDB.sql dosyasını SQL Server Management Studio dosyasında açıp Execute yaptığınız zaman Logni veritabanı ve içerisindeki bilgiler oluşacaktır
2.) ASP.NET MVC API ile oluşan Web API dosyası WebApplication1 dosyasıdır.  Bu projesi Visual Studio üzerinde açıp Google Chrome içerisinde çalıştırabilirsiniz.
3.) IONIC ile yapılmış Web+ Mobil uygulama ionic-http-get-app dosyasıdır. Bu klasörü Visual Studio Code üzerinde açınız. Visual Studio Code içerisindeki terminale ionic serve yazınız uygulama tarayıcı üzerinde çalışacaktır.
Google Chrome üzerinde F12 tuşuna basarak telefon simgesinden istediğiniz cihazı seçip cihazdaki görünümüme bakabilirsiniz.
4.) Oluşturulan API'yi test etmek için POSTMAN uygulamasını bilgisayarınıza yükleyiniz. Link : https://www.getpostman.com/downloads/
5.) Bir sonraki fazda PHP ile bir API yazılıp IONIC uygulamasında çalıştırılacaktır.
6.)Ardından apk oluşturularak bir telefonda denenecektir.
